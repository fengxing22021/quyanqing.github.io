麦克风权限被拒绝

您的浏览器拒绝了麦克风访问权限。请按照以下步骤解决此问题：

快速解决方法：

  1. 检查浏览器地址栏旁边的麦克风图标，点击并选择"允许"
  2. 刷新页面后再次尝试
  3. 确保没有其他标签页正在使用麦克风
Chrome
Firefox
Edge
Safari

Chrome 浏览器权限设置

  1. 点击浏览器地址栏右侧的锁形图标或麦克风图标
  2. 在弹出窗口中，将麦克风权限从"阻止"改为"允许"
  3. 如果看不到图标，请点击页面刷新按钮
  4. 或者前往 Chrome 设置 → 隐私和安全 → 网站设置 → 麦克风
  5. 找到当前网站并确保权限设置为"允许"

Firefox 浏览器权限设置

  1. 点击地址栏左侧的麦克风图标
  2. 选择"允许"以授予麦克风访问权限
  3. 或者前往 Firefox 选项 → 隐私与安全 → 权限 → 麦克风 → 设置
  4. 找到当前网站并设置为"允许"

Edge 浏览器权限设置

  1. 点击地址栏右侧的锁形图标或麦克风图标
  2. 将麦克风权限从"阻止"改为"允许"
  3. 或者前往 Edge 设置 → Cookie 和网站权限 → 麦克风
  4. 找到当前网站并设置为"允许"

Safari 浏览器权限设置

  1. Safari 偏好设置 → 网站 → 麦克风
  2. 找到当前网站并设置为"允许"
  3. 或者当网站请求麦克风权限时，点击"允许"

如果问题仍然存在：

  • 确保您的设备有麦克风并且工作正常
  • 检查操作系统级别的麦克风权限设置
  • 尝试使用其他浏览器
  • 确保网站使用 HTTPS 连接（本地环境除外）
